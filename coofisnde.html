Coofis NDE

UI/UX
Role
UI/UX Designer
Timeline
Feb 2024 – Jun 2024
Team
4 UI/UX Designer, 2 Graphic Designer
Tools
Figma, Microsoft Loop, Adobe Illustrator

Context

During my internship at PT ARM Solusi, my team and I worked on a project to redesign the Coofis application. Coofis is a platform developed to support collaboration and office correspondence management within organizations, providing an end-to-end solution for document workflows and internal communication. Our redesign focused on enhancing the overall user experience and interface consistency, while aligning the product with modern design principles and Coofis's visual identity.

Problem & Goals

The previous version of Coofis presented several usability and consistency issues that affected user efficiency and clarity. Many interface elements lacked structure, and the overall user flow felt fragmented. Inconsistent language use, unclear icons, and a long login process created friction for users in completing daily tasks.

  • Improve usability and interface consistency through a structured design system.
  • Simplify user flows, especially for login, 2FA, and document management.
  • Enhance visual hierarchy and readability by refining typography, spacing, and color use.
  • Ensure language consistency across all features to improve comprehension.
  • Align the design with Coofis's brand identity while maintaining a professional and minimal aesthetic.

Pain Point

Through a review of the previous Coofis interface and team discussions, we identified several key pain points that negatively impacted the user experience:

  • Unclear iconography
  • Complex login and 2FA process
  • Inconsistent language and copywriting
  • Lack of visual feedback
  • No Help or Report feature
  • Inconsistent layout and spacing

Research Insights

Before starting the redesign, our team aligned on the goals, constraints, and workflow to ensure efficiency and consistency throughout the project.

We aimed not only to fix usability issues but also to establish a new design system that reflected Coofis's identity and improved the user's overall journey. Our intention was to make the interface both functional and visually engaging, allowing users to enjoy smooth interactions while maintaining productivity.

Constraints

The project was focused on Coofis NDE (Nota Dinas Elektronik), one of the main applications within the Coofis ecosystem. The redesign process, from ideation to the finalized version 1.0, took approximately two months.

Workflow

With a relatively large team, we divided tasks into specific sub-divisions such as UX, UI, and Graphic Design. Each group contributed to shared milestones, ensuring that design decisions stayed consistent across the system. Collaboration was key, we continuously aligned on progress, exchanged feedback, and refined our outputs together.

Solution

Our redesign of Coofis NDE focused on creating a more consistent, intuitive, and modern interface while maintaining the professional identity of the platform. We approached the redesign with both usability and visual clarity in mind — ensuring that every improvement directly enhanced the user experience.

Key Improvements

  • Introduced a structured design system for consistency.
  • Simplified the login and 2FA flow.
  • Refined layout and hierarchy.
  • Unified language usage.
  • Enhanced visual indicators such as icons, tooltips, and states.
  • Implemented main components and auto-layouts.
  • Integrated illustration and iconography that align with Coofis's identity.

Added Features

To strengthen functionality and accessibility, we also introduced several new features in version 1.0 of the redesign:

  • Help & Report section
  • Error alert system
  • File upload with comment feature
  • Search list improvement
  • SMS-based 2FA
  • Password visibility toggle
  • Filter function

Results & Impact

The redesign of Coofis NDE successfully addressed the core usability and consistency issues found in the previous version. Through the creation of a unified design system and refined user flows, the product became more intuitive, visually coherent, and efficient for daily use.

Outcomes

  • Improved interface consistency
  • Enhanced user clarity
  • Better accessibility and readability
  • Streamlined workflows
  • Established a scalable design system

Beyond visual and functional improvements, this project also strengthened our team's collaboration and design communication, helping us better align with development teams and organizational stakeholders.

Learnings

The Coofis NDE redesign taught me the importance of collaboration, communication, and consistency in large scale projects. Working with a multidisciplinary team helped me understand how design decisions affect real implementation and user workflows. I also learned how a well-structured design system can improve efficiency and coherence across the product. Overall, this project strengthened my ability to think systematically, adapt to challenges, and design with both function and clarity in mind.
